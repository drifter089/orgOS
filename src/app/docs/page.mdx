# Documentation

Welcome to the project documentation! This documentation site is built with Next.js 15 and MDX.

## Features

- **MDX Support** - Write content in Markdown with React components
- **Syntax Highlighting** - Beautiful code blocks with `rehype-pretty-code`
- **Dark Mode** - Automatic theme switching support
- **GitHub Flavored Markdown** - Tables, task lists, and more

## Quick Start

Get started by exploring the documentation in the sidebar:

1. **Getting Started** - Learn the basics
2. **Components** - Explore available components
3. **API Reference** - Detailed API documentation
