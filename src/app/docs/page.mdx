## Core Technologies

Our carefully selected tech stack prioritizes developer experience, type safety, and performance:

```typescript
// The power of full-stack type safety
import { api } from "~/trpc/server";

// Server Component - Direct database call, no API needed
export default async function HomePage() {
  // Type-safe from database to UI
  const posts = await api.post.getLatest();

  return (
    <div>
      {posts.map(post => (
        <PostCard key={post.id} post={post} />
      ))}
    </div>
  );
}
```

## Why This Stack?

We've chosen each technology for specific reasons:

- **Next.js 15** - Server Components for performance, App Router for modern patterns
- **tRPC** - Eliminates API contracts, full type safety from database to UI
- **Prisma** - Type-safe database access with excellent DX
- **WorkOS** - Production-ready auth without the complexity
- **TanStack Query** - Best-in-class caching and synchronization
- **Tailwind CSS v4** - Modern styling with OKLCH color space

Learn more in our [Architectural Decisions](/docs/architecture/decisions) documentation.
